
## Cvent SDKs
Official Cvent SDK providing typed access to Cvent REST APIs for event management, contacts, attendees, and more using OAuth2 authentication.
For documentation, see the README and USAGE. To get credentials, visit https://developers.cvent.com/docs/rest-api/overview.
