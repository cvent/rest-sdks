# MutabilityJson

This is used to denote the mutability of the attribute.

## Example Usage

```csharp
using Cvent.SDK.Models.Components;

var value = MutabilityJson.ReadOnly;
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `ReadOnly`  | readOnly    |
| `ReadWrite` | readWrite   |
| `Immutable` | immutable   |
| `WriteOnly` | writeOnly   |
