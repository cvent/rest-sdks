# ParticipantResponseJson

Denotes a participant response to an appointment.

## Example Usage

```csharp
using Cvent.SDK.Models.Components;

var value = ParticipantResponseJson.Confirmed;
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `Confirmed` | CONFIRMED   |
| `Tentative` | TENTATIVE   |
| `Pending`   | PENDING     |
| `Declined`  | DECLINED    |
