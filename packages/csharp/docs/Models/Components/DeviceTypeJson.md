# DeviceTypeJson

The device used to watch the video. Videos can be viewed on desktop, mobile or tablet.

## Example Usage

```csharp
using Cvent.SDK.Models.Components;

var value = DeviceTypeJson.Desktop;
```


## Values

| Name      | Value     |
| --------- | --------- |
| `Desktop` | DESKTOP   |
| `Mobile`  | MOBILE    |
| `Tablet`  | TABLET    |
