# VideoTextTrackKindJson

Denotes the kind of video text track. Only subtitles are supported.

## Example Usage

```csharp
using Cvent.SDK.Models.Components;

var value = VideoTextTrackKindJson.Subtitles;
```


## Values

| Name           | Value          |
| -------------- | -------------- |
| `Subtitles`    | subtitles      |
| `Captions`     | captions       |
| `Descriptions` | descriptions   |
| `Chapters`     | chapters       |
| `Metadata`     | metadata       |
