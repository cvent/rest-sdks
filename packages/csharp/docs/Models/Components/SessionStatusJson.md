# SessionStatusJson

This is used to denote the status of a session.

## Example Usage

```csharp
using Cvent.SDK.Models.Components;

var value = SessionStatusJson.Active;
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `Active`    | Active      |
| `Cancelled` | Cancelled   |
