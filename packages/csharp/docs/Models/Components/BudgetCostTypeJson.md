# BudgetCostTypeJson

Denotes the cost type of a budget item. FIXED costs stay the same regardless of quantity. VARIABLE costs are based on a quantity.

## Example Usage

```csharp
using Cvent.SDK.Models.Components;

var value = BudgetCostTypeJson.Fixed;
```


## Values

| Name       | Value      |
| ---------- | ---------- |
| `Fixed`    | FIXED      |
| `Variable` | VARIABLE   |
