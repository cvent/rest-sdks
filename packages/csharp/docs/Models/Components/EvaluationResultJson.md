# EvaluationResultJson

This represents the evaluation status of an assessment chapter.

## Example Usage

```csharp
using Cvent.SDK.Models.Components;

var value = EvaluationResultJson.Passed;
```


## Values

| Name          | Value         |
| ------------- | ------------- |
| `Passed`      | Passed        |
| `Failed`      | Failed        |
| `Unevaluated` | Unevaluated   |
