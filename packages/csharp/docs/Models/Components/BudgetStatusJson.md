# BudgetStatusJson

Denotes the status assigned to a budget item.

## Example Usage

```csharp
using Cvent.SDK.Models.Components;

var value = BudgetStatusJson.Estimated;
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `Estimated` | ESTIMATED   |
| `Requested` | REQUESTED   |
| `Pending`   | PENDING     |
| `Confirmed` | CONFIRMED   |
| `Invoiced`  | INVOICED    |
| `Paid`      | PAID        |
