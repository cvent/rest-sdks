# CardStatusJson

This is used to denote the card status for a card.

## Example Usage

```csharp
using Cvent.SDK.Models.Components;

var value = CardStatusJson.Pending;
```

| ----------- | ----------- |
| `Pending`   | Pending     |
| `Active`    | Active      |
| `Inactive`  | Inactive    |

| `Cancelled` | Cancelled   |
