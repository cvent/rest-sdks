# AgeQualifyingCode

Code that specifies if the guest is an adult or child. (10 - adult, 8 - child)

## Example Usage

```csharp
using Cvent.SDK.Models.Components;

var value = AgeQualifyingCode.Ten;
```


## Values

| Name    | Value   |
| ------- | ------- |
| `Ten`   | 10      |
| `Eight` | 8       |
