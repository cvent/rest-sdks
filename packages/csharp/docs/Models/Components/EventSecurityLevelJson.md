# EventSecurityLevelJson

Represents the security level used for event registrations.

## Example Usage

```csharp
using Cvent.SDK.Models.Components;

var value = EventSecurityLevelJson.Private;
```


## Values

| Name               | Value              |
| ------------------ | ------------------ |
| `Private`          | Private            |
| `Public`           | Public             |
| `PublicAndPrivate` | Public and Private |
