# TravelProgramStayTypeJson

Code representing the stay types requested by the program.  Programs can accept daily only, or daily and extended stay proposals.

## Example Usage

```typescript
import { TravelProgramStayTypeJson } from "@cvent/sdk/models/components";

let value: TravelProgramStayTypeJson = "daily";
```

## Values

```typescript
"daily" | "daily_and_extended_stay"
```
