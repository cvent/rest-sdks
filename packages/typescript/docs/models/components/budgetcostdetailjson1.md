# BudgetCostDetailJson1

The budget entry in detail, which consists of the budget category or sub-category, the number of units, the cost per unit, and the total amount.

## Example Usage

```typescript
import { BudgetCostDetailJson1 } from "@cvent/sdk/models/components";

let value: BudgetCostDetailJson1 = {
  category: {
    id: 3,
    name: "Travel",
  },
  subCategory: {
    id: "04ca6ae2-0dc3-487b-953e-86d6abbdf7d3",
    name: "utilities",
  },
  units: 5.2,
  cost: 100.5,
  totalCost: 100.5,
};
```
