# DateTimeDisplayFormatJson

This is used to denote the display format type of date time type custom field.

## Example Usage

```typescript
import { DateTimeDisplayFormatJson } from "@cvent/sdk/models/components";

let value: DateTimeDisplayFormatJson = "DateOnlyMonthFirst";
```

## Values

```typescript
"DateAndTimeMonthFirst12HourTime" | "DateAndTimeMonthFirst24HourTime" | "DateOnlyMonthFirst" | "DateAndTimeDayFirst12HourTime" | "DateAndTimeDayFirst24HourTime" | "DateOnlyDayFirst"
```
