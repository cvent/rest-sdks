# AppointmentTypeJson

This is used for storing details of an appointment type.

## Example Usage

```typescript
import { AppointmentTypeJson } from "@cvent/sdk/models/components";

let value: AppointmentTypeJson = {
  id: "0046e875-79a5-4851-8332-da105d58088a",
  name: "Event Appointment",
  category: "CUSTOM_APPOINTMENT",
  timeRangeType: "DEFINED",
};
```
