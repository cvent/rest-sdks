# BudgetCostTypeJson

Denotes the cost type of a budget item. FIXED costs stay the same regardless of quantity. VARIABLE costs are based on a quantity.

## Example Usage

```typescript
import { BudgetCostTypeJson } from "@cvent/sdk/models/components";

let value: BudgetCostTypeJson = "FIXED";
```

## Values

```typescript
"FIXED" | "VARIABLE"
```
