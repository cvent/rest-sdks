# BudgetStatusJson

Denotes the status assigned to a budget item.

## Example Usage

```typescript
import { BudgetStatusJson } from "@cvent/sdk/models/components";

let value: BudgetStatusJson = "ESTIMATED";
```

## Values

```typescript
"ESTIMATED" | "REQUESTED" | "PENDING" | "CONFIRMED" | "INVOICED" | "PAID"
```
