# OpenEndedOneLine

This is used to denote the open ended one line type of custom field.

## Example Usage

```typescript
import { OpenEndedOneLine } from "@cvent/sdk/models/components";

let value: OpenEndedOneLine = {
  answerFormat: "General",
  minimum: 2,
  maximum: 8,
};
```
