# BudgetVendorJson

List of details for a vendor assigned to a budget item.

## Example Usage

```typescript
import { BudgetVendorJson } from "@cvent/sdk/models/components";

let value: BudgetVendorJson = {
  id: "d64380fd-3631-43e9-aac7-bd6bb6eccf6b",
  name: "Best Rest Hotels",
  type: "CSN",
};
```
