# AppointmentAvailabilityTypeJson

Denotes whether an availability time range represents when an attendee is available or unavailable.

## Example Usage

```typescript
import { AppointmentAvailabilityTypeJson } from "@cvent/sdk/models/components";

let value: AppointmentAvailabilityTypeJson = "AVAILABLE";
```

## Values

```typescript
"AVAILABLE" | "UNAVAILABLE"
```
