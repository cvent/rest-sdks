# PublicUsagePlan

Object representing an API Platform Usage Tier, including information when the usage plan is scheduled to revert to a previous plan, if any.

## Example Usage

```typescript
import { PublicUsagePlan } from "@cvent/sdk/models/components";

let value: PublicUsagePlan = {
  name: "API_Platform_Free_Tier",
  quota: 1000,
  burstLimit: 1,
  rateLimit: 2,
  switchBackPlan: {
    name: "API_Platform_Standard_Tier",
    time: new Date("2023-10-01T12:00:00Z"),
  },
};
```
