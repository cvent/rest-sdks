# EventQuestionChoiceJson

This is used to denote the choice of questions in an event.

## Example Usage

```typescript
import { EventQuestionChoiceJson } from "@cvent/sdk/models/components";

let value: EventQuestionChoiceJson = {
  text: "What is your current designation?",
};
```
