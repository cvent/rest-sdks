# ConflictResolution

Represents choices in resolving conflicts in the contacts to be merged or to optionally select fields to keep from a secondary contact, overriding default behavior.

## Example Usage

```typescript
import { ConflictResolution } from "@cvent/sdk/models/components";

let value: ConflictResolution = {
  contactFields: [
    {
      fieldName: "firstName",
      contactId: "a45a3341-d30f-4bd9-bfd8-fbcdae37f0f2",
    },
  ],
};
```
