# ProductJson

Represents a product. Products are items that can be purchased during registration.

## Example Usage

```typescript
import { ProductJson } from "@cvent/sdk/models/components";

let value: ProductJson = {
  id: "a64b648e-b7b0-4031-8f8a-8d5e0919b936",
  type: "AdmissionItem",
  name: "Admission Item 1",
};
```
