# HubColorsJson

All the colors associated with the Events+ hub.

## Example Usage

```typescript
import { HubColorsJson } from "@cvent/sdk/models/components";

let value: HubColorsJson = {
  action: "#0F0F0F",
  background: "#0F0F0F",
  main: "#0F0F0F",
  mood: "#0F0F0F",
};
```
