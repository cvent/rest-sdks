# MeetingRoomRequirementJson

The requirement of a meeting room on the specified date.

## Example Usage

```typescript
import { MeetingRoomRequirementJson } from "@cvent/sdk/models/components";

let value: MeetingRoomRequirementJson = {
  date: "2099-12-11",
  startTime: "09:30:00",
  endTime: "14:45:00",
  agendaItem: "Meeting Room Agenda point 1.",
  agendaItemType: [
    "Breakout",
  ],
  setupType: [
    "Classroom",
  ],
  roomSize: 100,
  numberOfPeople: 250,
  twentyFourHourHold: false,
  note: "Meeting room note.",
};
```
