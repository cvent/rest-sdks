# QuestionSectionJson

Code representing the section where the question belongs. Currently, program_client (client specific) and program_g_and_m (group and meeting) are supported.

## Example Usage

```typescript
import { QuestionSectionJson } from "@cvent/sdk/models/components";

let value: QuestionSectionJson = "program_client";
```

## Values

```typescript
"program_client" | "program_g_and_m"
```
