# QuantityItem

Represents a quantity item. A quantity item allows invitees to specify the amount of a particular product, such as raffle tickets, t-shirts, or tote bags, they would like to purchase.

## Example Usage

```typescript
import { QuantityItem } from "@cvent/sdk/models/components";
import { RFCDate } from "@cvent/sdk/types";

let value: QuantityItem = {
  registrantInformation: "Registrant Note.",
  id: "04ca6ae2-0dc3-487b-953e-86d6abbdf7d3",
  name: "General Quantity Item",
  description: "Some description of the quantity item.",
  code: "R5NP4NJYB3D",
  maximumItemsPerInvitee: 10,
  registrationTypes: [
    "04ca6ae2-0dc3-487b-953e-86d6abbdf7d4",
  ],
  openForRegistration: false,
  automaticallyClosesOn: new RFCDate("2022-08-12"),
  capacity: 150,
  closeEventOnCapacityFull: false,
};
```
