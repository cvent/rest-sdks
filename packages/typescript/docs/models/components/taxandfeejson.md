# TaxAndFeeJson

A single tax or fee applicable to a room.

## Example Usage

```typescript
import { TaxAndFeeJson } from "@cvent/sdk/models/components";

let value: TaxAndFeeJson = {
  name: "State Tax",
  amount: 6,
  amountType: "percentage",
  frequency: "per_room",
  partOfBaseRate: false,
  partOfDisplayRate: false,
};
```
