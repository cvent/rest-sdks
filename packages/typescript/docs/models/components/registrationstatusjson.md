# RegistrationStatusJson

This is used to denote the registration status for an event.

## Example Usage

```typescript
import { RegistrationStatusJson } from "@cvent/sdk/models/components";

let value: RegistrationStatusJson = "Pending";
```

## Values

```typescript
"Active" | "Cancelled" | "Closed" | "Completed" | "Deleted" | "NoRegistrationRequired" | "Pending" | "Processing"
```
