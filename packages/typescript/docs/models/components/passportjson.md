# PassportJson

This entity represents a contact's passport information.

## Example Usage

```typescript
import { PassportJson } from "@cvent/sdk/models/components";

let value: PassportJson = {
  number: "123456789",
  country: "United Kingdom",
  countryCode: "GB",
};
```
