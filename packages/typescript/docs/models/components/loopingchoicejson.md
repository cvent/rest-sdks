# LoopingChoiceJson

Loop Choice is the choice for which the chapter is looped. Questions in the looping chapter get repeated once for each choice selected in the parent question

## Example Usage

```typescript
import { LoopingChoiceJson } from "@cvent/sdk/models/components";

let value: LoopingChoiceJson = {
  id: "9463c74e-18c6-401a-a710-ae0f485bf059",
};
```
