# RefundPolicyJson

Refund policy associated to a fee.

## Example Usage

```typescript
import { RefundPolicyJson } from "@cvent/sdk/models/components";
import { RFCDate } from "@cvent/sdk/types";

let value: RefundPolicyJson = {
  amount: 50,
  percentage: 50,
  type: "RefundByPercentage",
  cancelByDate: new RFCDate("2020-02-09"),
};
```
