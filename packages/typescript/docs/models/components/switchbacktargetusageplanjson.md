# SwitchBackTargetUsagePlanJson

Object representing the target usage plan and the date when the current plan is scheduled to revert to it. This object is only present when a switch-back is scheduled.

## Example Usage

```typescript
import { SwitchBackTargetUsagePlanJson } from "@cvent/sdk/models/components";

let value: SwitchBackTargetUsagePlanJson = {
  name: "API_Platform_Standard_Tier",
  time: new Date("2023-10-01T12:00:00Z"),
};
```
