# ThumbnailTypeJson

This is used to denote the file type of a thumbnail

## Example Usage

```java
import com.cvent.models.components.ThumbnailTypeJson;

ThumbnailTypeJson value = ThumbnailTypeJson.BMP;
```


## Values

| Name   | Value  |
| ------ | ------ |
| `BMP`  | bmp    |
| `GIF`  | gif    |
| `JPE`  | jpe    |
| `JPEG` | jpeg   |
| `JPG`  | jpg    |
| `PNG`  | png    |
| `SVG`  | svg    |
