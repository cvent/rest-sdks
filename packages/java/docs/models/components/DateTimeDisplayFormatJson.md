# DateTimeDisplayFormatJson

This is used to denote the display format type of date time type custom field.

## Example Usage

```java
import com.cvent.models.components.DateTimeDisplayFormatJson;

DateTimeDisplayFormatJson value = DateTimeDisplayFormatJson.DATE_AND_TIME_MONTH_FIRST12_HOUR_TIME;
```


## Values

| Name                                    | Value                                   |
| --------------------------------------- | --------------------------------------- |
| `DATE_AND_TIME_MONTH_FIRST12_HOUR_TIME` | DateAndTimeMonthFirst12HourTime         |
| `DATE_AND_TIME_MONTH_FIRST24_HOUR_TIME` | DateAndTimeMonthFirst24HourTime         |
| `DATE_ONLY_MONTH_FIRST`                 | DateOnlyMonthFirst                      |
| `DATE_AND_TIME_DAY_FIRST12_HOUR_TIME`   | DateAndTimeDayFirst12HourTime           |
| `DATE_AND_TIME_DAY_FIRST24_HOUR_TIME`   | DateAndTimeDayFirst24HourTime           |
| `DATE_ONLY_DAY_FIRST`                   | DateOnlyDayFirst                        |
