# ParticipantTypeJson

Denotes if a participant is a host or attendee of an appointment.

## Example Usage

```java
import com.cvent.models.components.ParticipantTypeJson;

ParticipantTypeJson value = ParticipantTypeJson.HOST;
```


## Values

| Name       | Value      |
| ---------- | ---------- |
| `HOST`     | Host       |
| `ATTENDEE` | Attendee   |
