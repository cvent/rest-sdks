# HotelRequestJsonRequesterType

Attendee type of the hotel requester. Contact means the requester is an independent attendee. Guest means the attendee is accompanying another attendee.

## Example Usage

```java
import com.cvent.models.components.HotelRequestJsonRequesterType;

HotelRequestJsonRequesterType value = HotelRequestJsonRequesterType.CONTACT;
```


## Values

| Name      | Value     |
| --------- | --------- |
| `CONTACT` | Contact   |
| `GUEST`   | Guest     |
