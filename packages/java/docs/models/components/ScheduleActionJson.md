# ScheduleActionJson

This is used to denote the action to perform with the scheduled task

## Example Usage

```java
import com.cvent.models.components.ScheduleActionJson;

ScheduleActionJson value = ScheduleActionJson.PROVISION;
```


## Values

| Name         | Value        |
| ------------ | ------------ |
| `PROVISION`  | provision    |
| `ACTIVATE`   | activate     |
| `DEACTIVATE` | deactivate   |
| `ARCHIVE`    | archive      |
