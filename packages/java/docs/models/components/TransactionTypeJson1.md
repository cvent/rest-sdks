# TransactionTypeJson1

This is used to denote the transaction type for a transaction.

## Example Usage

```java
import com.cvent.models.components.TransactionTypeJson1;

TransactionTypeJson1 value = TransactionTypeJson1.PAYMENTS;
```


## Values

| Name         | Value        |
| ------------ | ------------ |
| `PAYMENTS`   | Payments     |
| `REFUNDS`    | Refunds      |
| `AUTHORIZED` | Authorized   |
| `APPROVED`   | Approved     |
| `DECLINED`   | Declined     |
