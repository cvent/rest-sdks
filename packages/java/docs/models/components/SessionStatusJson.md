# SessionStatusJson

This is used to denote the status of a session.

## Example Usage

```java
import com.cvent.models.components.SessionStatusJson;

SessionStatusJson value = SessionStatusJson.ACTIVE;
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `ACTIVE`    | Active      |
| `CANCELLED` | Cancelled   |
| `CLOSED`    | Closed      |
