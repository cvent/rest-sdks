# RegistrationStatusJson

This is used to denote the registration status for an event.


## Values

| Name                       | Value                      |
| -------------------------- | -------------------------- |
| `ACTIVE`                   | Active                     |
| `CANCELLED`                | Cancelled                  |
| `CLOSED`                   | Closed                     |
| `COMPLETED`                | Completed                  |
| `DELETED`                  | Deleted                    |
| `NO_REGISTRATION_REQUIRED` | NoRegistrationRequired     |
| `PENDING`                  | Pending                    |
| `PROCESSING`               | Processing                 |
