# AttendeeVisibilityJson

Denotes the visibility of the attendee profile to other attendees. Private: Their profile is not visible. Public: Their profile is visible.

## Example Usage

```java
import com.cvent.models.components.AttendeeVisibilityJson;

AttendeeVisibilityJson value = AttendeeVisibilityJson.PRIVATE;
```


## Values

| Name      | Value     |
| --------- | --------- |
| `PRIVATE` | Private   |
| `PUBLIC`  | Public    |
