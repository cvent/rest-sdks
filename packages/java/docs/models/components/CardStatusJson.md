# CardStatusJson

This is used to denote the card status for a card.

## Example Usage

```java
import com.cvent.models.components.CardStatusJson;

CardStatusJson value = CardStatusJson.PENDING;
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `PENDING`   | Pending     |
| `ACTIVE`    | Active      |
| `INACTIVE`  | Inactive    |
| `EXPIRED`   | Expired     |
| `CANCELLED` | Cancelled   |
