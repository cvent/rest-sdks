# SendEmailStatusJson

It is used to denote the status of the sent email.

## Example Usage

```java
import com.cvent.models.components.SendEmailStatusJson;

SendEmailStatusJson value = SendEmailStatusJson.SENT;
```


## Values

| Name          | Value         |
| ------------- | ------------- |
| `SENT`        | Sent          |
| `IN_PROGRESS` | In-Progress   |
| `ERROR`       | Error         |
| `OPTED_OUT`   | Opted-Out     |
