# ParticipantResponseJson

Denotes a participant response to an appointment.

## Example Usage

```java
import com.cvent.models.components.ParticipantResponseJson;

ParticipantResponseJson value = ParticipantResponseJson.CONFIRMED;
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `CONFIRMED` | CONFIRMED   |
| `TENTATIVE` | TENTATIVE   |
| `PENDING`   | PENDING     |
| `DECLINED`  | DECLINED    |
