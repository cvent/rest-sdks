# BudgetCostTypeJson

Denotes the cost type of a budget item. FIXED costs stay the same regardless of quantity. VARIABLE costs are based on a quantity.

## Example Usage

```java
import com.cvent.models.components.BudgetCostTypeJson;

BudgetCostTypeJson value = BudgetCostTypeJson.FIXED;
```


## Values

| Name       | Value      |
| ---------- | ---------- |
| `FIXED`    | FIXED      |
| `VARIABLE` | VARIABLE   |
