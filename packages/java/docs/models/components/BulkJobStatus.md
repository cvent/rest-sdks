# BulkJobStatus

The current status of the bulk job.

## Example Usage

```java
import com.cvent.models.components.BulkJobStatus;

BulkJobStatus value = BulkJobStatus.PENDING;
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `PENDING`   | PENDING     |
| `READY`     | READY       |
| `RUNNING`   | RUNNING     |
| `CANCELED`  | CANCELED    |
| `COMPLETED` | COMPLETED   |
| `FAILED`    | FAILED      |
