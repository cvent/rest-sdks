# BadgePrintJobStatusJson

This is used to indicate the status of the badge print job.

## Example Usage

```java
import com.cvent.models.components.BadgePrintJobStatusJson;

BadgePrintJobStatusJson value = BadgePrintJobStatusJson.SCHEDULED;
```


## Values

| Name          | Value         |
| ------------- | ------------- |
| `SCHEDULED`   | Scheduled     |
| `RESCHEDULED` | Rescheduled   |
| `ASSIGNED`    | Assigned      |
| `FAILED`      | Failed        |
| `DONE`        | Done          |
| `CANCELED`    | Canceled      |
