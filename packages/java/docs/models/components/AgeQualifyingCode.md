# AgeQualifyingCode

Code that specifies if the guest is an adult or child. (10 - adult, 8 - child)

## Example Usage

```java
import com.cvent.models.components.AgeQualifyingCode;

AgeQualifyingCode value = AgeQualifyingCode.TEN;
```


## Values

| Name    | Value   |
| ------- | ------- |
| `TEN`   | 10      |
| `EIGHT` | 8       |
