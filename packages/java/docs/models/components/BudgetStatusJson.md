# BudgetStatusJson

Denotes the status assigned to a budget item.

## Example Usage

```java
import com.cvent.models.components.BudgetStatusJson;

BudgetStatusJson value = BudgetStatusJson.ESTIMATED;
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `ESTIMATED` | ESTIMATED   |
| `REQUESTED` | REQUESTED   |
| `PENDING`   | PENDING     |
| `CONFIRMED` | CONFIRMED   |
| `INVOICED`  | INVOICED    |
| `PAID`      | PAID        |
