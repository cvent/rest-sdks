# AppointmentAvailabilityTypeJson

Denotes whether an availability time range represents when an attendee is available or unavailable.

## Example Usage

```java
import com.cvent.models.components.AppointmentAvailabilityTypeJson;

AppointmentAvailabilityTypeJson value = AppointmentAvailabilityTypeJson.AVAILABLE;
```


## Values

| Name          | Value         |
| ------------- | ------------- |
| `AVAILABLE`   | AVAILABLE     |
| `UNAVAILABLE` | UNAVAILABLE   |
