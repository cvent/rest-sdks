# AttendeeCreditTypeJson

Represents the entity that granted the credit. Event: This credit was granted by an event. Session: This credit was granted by a session.

## Example Usage

```java
import com.cvent.models.components.AttendeeCreditTypeJson;

AttendeeCreditTypeJson value = AttendeeCreditTypeJson.EVENT;
```


## Values

| Name      | Value     |
| --------- | --------- |
| `EVENT`   | Event     |
| `SESSION` | Session   |
