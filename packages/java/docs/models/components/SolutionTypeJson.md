# SolutionTypeJson

This is used to denote the solution type to be used for webcast. The 'Cvent Video Conferencing' refers to the video experience which links out the Attendee hub whereas 'Embedded Cvent Video Conferencing' refers to the video experience within the Attendee hub.


## Values

| Name                                | Value                               |
| ----------------------------------- | ----------------------------------- |
| `CVENT_VIDEO_CONFERENCING`          | Cvent Video Conferencing            |
| `EXTERNAL_PLATFORM`                 | External Platform                   |
| `CVENT_STUDIO`                      | Cvent Studio                        |
| `EMBEDDED_CVENT_VIDEO_CONFERENCING` | Embedded Cvent Video Conferencing   |
| `CVENT_STUDIO_LITE`                 | Cvent Studio Lite                   |
| `EMBEDDED_CVENT_STUDIO`             | Embedded Cvent Studio               |
