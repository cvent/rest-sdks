# WebcastTypeJson

This is used to denote the type of a webcast

## Example Usage

```java
import com.cvent.models.components.WebcastTypeJson;

WebcastTypeJson value = WebcastTypeJson.MEETING;
```


## Values

| Name           | Value          |
| -------------- | -------------- |
| `MEETING`      | Meeting        |
| `WEBINAR`      | Webinar        |
| `PRESENTATION` | Presentation   |
