# FairSeasonTypeJson

Fair season type. Blackout means that no special rate is negotiated for that date and Best Available Rates will apply

## Example Usage

```java
import com.cvent.models.components.FairSeasonTypeJson;

FairSeasonTypeJson value = FairSeasonTypeJson.NEGOTIATED;
```


## Values

| Name         | Value        |
| ------------ | ------------ |
| `NEGOTIATED` | negotiated   |
| `BLACKOUT`   | blackout     |
