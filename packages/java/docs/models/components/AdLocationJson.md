# AdLocationJson

The placement of the ad within the specific product.

## Example Usage

```java
import com.cvent.models.components.AdLocationJson;

AdLocationJson value = AdLocationJson.HOME_PAGE;
```


## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `HOME_PAGE`      | Home Page        |
| `EXHIBITOR_LIST` | Exhibitor List   |
| `SESSION_LIST`   | Session List     |
