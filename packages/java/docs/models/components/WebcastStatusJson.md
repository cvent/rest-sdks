# WebcastStatusJson

This is used to denote the status of a webcast

## Example Usage

```java
import com.cvent.models.components.WebcastStatusJson;

WebcastStatusJson value = WebcastStatusJson.NOT_STARTED;
```


## Values

| Name          | Value         |
| ------------- | ------------- |
| `NOT_STARTED` | Not Started   |
| `STARTED`     | Started       |
| `FINISHED`    | Finished      |
| `CANCELLED`   | Cancelled     |
