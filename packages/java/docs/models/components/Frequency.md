# Frequency

Defines how often the tax or fee is applied.
- per_guest: Applied per guest.
- per_room: Applied per room.
- per_night: Applied per night.
- per_stay: Applied per stay.

## Example Usage

```java
import com.cvent.models.components.Frequency;

Frequency value = Frequency.PER_GUEST;
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `PER_GUEST` | per_guest   |
| `PER_ROOM`  | per_room    |
| `PER_NIGHT` | per_night   |
| `PER_STAY`  | per_stay    |
