# ParticipantStatusJson

Denotes the status of a participant in an appointment.


## Values

| Name               | Value              |
| ------------------ | ------------------ |
| `PENDING`          | pending            |
| `PENDING_PROPOSED` | pending_proposed   |
| `CONFIRMED`        | confirmed          |
| `DECLINED`         | declined           |
| `CANCELLED`        | cancelled          |
| `TENTATIVE`        | tentative          |
