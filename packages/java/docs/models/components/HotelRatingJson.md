# HotelRatingJson

The hotel's rating, used to classify the hotel's quality.


## Values

| Name         | Value        |
| ------------ | ------------ |
| `NA`         | N/A          |
| `BUDGET`     | Budget       |
| `ONE_STAR`   | 1 Star       |
| `TWO_STAR`   | 2 Star       |
| `THREE_STAR` | 3 Star       |
| `FOUR_STAR`  | 4 Star       |
| `FIVE_STAR`  | 5 Star       |
