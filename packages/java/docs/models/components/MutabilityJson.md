# MutabilityJson

This is used to denote the mutability of the attribute.

## Example Usage

```java
import com.cvent.models.components.MutabilityJson;

MutabilityJson value = MutabilityJson.READ_ONLY;
```


## Values

| Name         | Value        |
| ------------ | ------------ |
| `READ_ONLY`  | readOnly     |
| `READ_WRITE` | readWrite    |
| `IMMUTABLE`  | immutable    |
| `WRITE_ONLY` | writeOnly    |
