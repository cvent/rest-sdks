# FormatJson

This is used to denote the format of a webcast

## Example Usage

```java
import com.cvent.models.components.FormatJson;

FormatJson value = FormatJson.LIVE;
```


## Values

| Name           | Value          |
| -------------- | -------------- |
| `LIVE`         | Live           |
| `PRE_RECORDED` | Pre-recorded   |
