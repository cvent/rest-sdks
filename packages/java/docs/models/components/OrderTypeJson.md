# OrderTypeJson

This is used to denote the type of order made by an attendee.

| Name             | Value            |
| ---------------- | ---------------- |
| `AUTHORIZATION`  | Authorization    |
| `ONLINE_CHARGE`  | Online Charge    |
| `ONLINE_REFUND`  | Online Refund    |
| `OFFLINE_CHARGE` | Offline Charge   |
| `OFFLINE_REFUND` | Offline Refund   |
