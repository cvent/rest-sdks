# AlternateTravelDetailJsonRequesterType

Attendee type of the entity this alternate travel belongs to. Contact means the requester is an independent attendee. Guest means the attendee is accompanying another attendee.

## Example Usage

```java
import com.cvent.models.components.AlternateTravelDetailJsonRequesterType;

AlternateTravelDetailJsonRequesterType value = AlternateTravelDetailJsonRequesterType.CONTACT;
```


## Values

| Name      | Value     |
| --------- | --------- |
| `CONTACT` | Contact   |
| `GUEST`   | Guest     |
