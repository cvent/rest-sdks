# ReconciliationStatusJson

This is used to denote the reconciliation status for a transaction.

## Example Usage

```java
import com.cvent.models.components.ReconciliationStatusJson;

ReconciliationStatusJson value = ReconciliationStatusJson.RECONCILED;
```


## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `RECONCILED`     | Reconciled       |
| `NOT_RECONCILED` | Not Reconciled   |
