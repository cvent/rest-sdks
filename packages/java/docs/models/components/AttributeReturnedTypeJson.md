# AttributeReturnedTypeJson

This is used to denote the returned type of the attribute.

## Example Usage

```java
import com.cvent.models.components.AttributeReturnedTypeJson;

AttributeReturnedTypeJson value = AttributeReturnedTypeJson.ALWAYS;
```


## Values

| Name      | Value     |
| --------- | --------- |
| `ALWAYS`  | always    |
| `NEVER`   | never     |
| `DEFAULT` | default   |
| `REQUEST` | request   |
