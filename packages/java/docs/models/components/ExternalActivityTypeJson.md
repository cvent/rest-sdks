# ExternalActivityTypeJson

This is used to denote the type of the external attendee activity.


## Values

| Name                   | Value                  |
| ---------------------- | ---------------------- |
| `EVENT_EXTERNAL`       | event.external         |
| `SESSION_EXTERNAL`     | session.external       |
| `APPOINTMENT_EXTERNAL` | appointment.external   |
| `EXHIBITOR_EXTERNAL`   | exhibitor.external     |
