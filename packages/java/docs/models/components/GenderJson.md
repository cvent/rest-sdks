# GenderJson

The gender of the contact.

## Example Usage

```java
import com.cvent.models.components.GenderJson;

GenderJson value = GenderJson.MALE;
```


## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `MALE`           | Male             |
| `FEMALE`         | Female           |
| `NON_BINARY`     | Non-binary       |
| `RATHER_NOT_SAY` | Rather not say   |
