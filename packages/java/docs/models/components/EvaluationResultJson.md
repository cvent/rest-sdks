# EvaluationResultJson

This represents the evaluation status of an assessment chapter.

## Example Usage

```java
import com.cvent.models.components.EvaluationResultJson;

EvaluationResultJson value = EvaluationResultJson.PASSED;
```


## Values

| Name          | Value         |
| ------------- | ------------- |
| `PASSED`      | Passed        |
| `FAILED`      | Failed        |
| `UNEVALUATED` | Unevaluated   |
