# ErrorScimTypeJson

This is used to denote the scim type of the error.

## Example Usage

```java
import com.cvent.models.components.ErrorScimTypeJson;

ErrorScimTypeJson value = ErrorScimTypeJson.INVALID_FILTER;
```


## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `INVALID_FILTER` | invalidFilter    |
| `UNIQUENESS`     | uniqueness       |
| `INVALID_SYNTAX` | invalidSyntax    |
| `INVALID_VALUE`  | invalidValue     |
