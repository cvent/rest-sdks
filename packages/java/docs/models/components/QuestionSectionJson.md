# QuestionSectionJson

Code representing the section where the question belongs. Currently, program_client (client specific) and program_g_and_m (group and meeting) are supported.

## Example Usage

```java
import com.cvent.models.components.QuestionSectionJson;

QuestionSectionJson value = QuestionSectionJson.PROGRAM_CLIENT;
```


## Values

| Name              | Value             |
| ----------------- | ----------------- |
| `PROGRAM_CLIENT`  | program_client    |
| `PROGRAM_G_AND_M` | program_g_and_m   |
