# LaunchFeatureTypeJson

These are the types of features used in the event container events that can be launched.

## Example Usage

```java
import com.cvent.models.components.LaunchFeatureTypeJson;

LaunchFeatureTypeJson value = LaunchFeatureTypeJson.WEBSITE;
```


## Values

| Name           | Value          |
| -------------- | -------------- |
| `WEBSITE`      | Website        |
| `REGISTRATION` | Registration   |
